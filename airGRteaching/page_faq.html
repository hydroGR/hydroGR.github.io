Frequently asked questions

Are all the airGR models available in airGRteahing?

  • Yes, all the airGR models from annual to hourly time-steps are available in airGRteaching. But its Shiny graphical user interface only run the monthly model (GR2M) and the three daily models (GR4J, GR5J and GR6J). If you want the run GR1A, GR4H or GR5H, you have to use the three simple functions.

Is the CemaNeige model available in airGRteaching?

  • Yes either with the three functions or with the Shiny interface, but it is automatically coupled to the hydrological models. The user cannot run this snow model alone.

Can I load many dataset in the Shiny interface?

  • Yes. You just have to put the different data.frame in a list. If you want to use different simulation period for each dataset, you just have to define it as a list of periods (one period is a vector of 2 values to define the beginning and end of the warm-up period [“YYYY-mm-dd” or “YYYY-mm-dd HH:MM:SS”]).

Do I have to launch the graphical user interface from R on my own personal computer?

  • No, you can also run the airGRteaching Shiny GUI on the INRAE Sunshine platform, but you cannot use your own datasets. To log in to the website, you just have to use the user name and the password that you use in you own institution.

I can’t find exactly the same results in the airGRteaching interface when I perform an automatic calibration compared to what I found in airGR. How is that possible?

  • In the interface, the range of the parameters is limited. It is very likely the reason why you find different parameter values.

Do you provide modelling exercises that I can use for my lectures?

  • No, but exercises will be available very soon. Stay aware!
